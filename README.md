# REAL-TIME---COLLABORATION-TOOL

## Description

A collaborative note-taking application similar to Google Docs.

## Features

- Real-Time Collaboration
- Multi-User Support
- Shared Notes
- WebSocket Communication
- Instant Synchronization

## Installation

### Backend

cd server

npm install

npm start

Server runs on:

http://localhost:5000

### Frontend

Open client/index.html in browser.

## Technologies

- HTML
- CSS
- JavaScript
- Node.js
- Socket.IO
